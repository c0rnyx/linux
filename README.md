# 🐧 Linux Practice — OverTheWire Bandit

This repository documents my hands-on Linux practice through the **OverTheWire Bandit wargame**. The goal of this project is to build a strong foundation in Linux system navigation, command-line tools, and basic security concepts by solving progressively challenging levels.

## 📚 About Bandit
The **Bandit wargame** from [OverTheWire](https://overthewire.org/wargames/bandit/) is designed for beginners who want to learn the fundamentals of Linux and security. Each level requires discovering the password for the next level by exploring the system and using common Linux utilities.

## 🎯 Objectives
- Strengthen **Linux command-line proficiency**
- Learn how to **navigate filesystems efficiently**
- Practice using essential tools
- Develop a **problem-solving mindset** commonly used in cybersecurity and penetration testing.

## 📂 Repository Structure
Each level contains notes, commands used, and explanations of the solution approach.

## 🧠 Learning Approach
For every level I documented:
- The **challenge description**
- **Commands and techniques** used
- **Key takeaway** or concept learned

The goal is not just solving the challenge but understanding **why the solution works**.

## 🛠 Skills Practiced
- Linux filesystem navigation  
- File inspection and analysis  
- Text processing  
- Data encoding/decoding  
- Permissions and file access  
- Basic security mindset  

## 🚀 My takeaways after finishing the challenge
The Bandit challenges helped me build the **core Linux skills required for cybersecurity**, penetration testing, and system administration in a 'fun but also challenging' way that focused mainly on hands-on approach.
